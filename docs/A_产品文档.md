# 创作工坊（AI Content Creator）产品文档

## 一、产品概述

创作工坊（AI Content Creator）是一款面向内容创作者的 AI 辅助排版与创作工具。产品的整体视觉风格定位为「优雅精致（Elegant & Refined）」，参考了小红书等平台上高质量图文的排版风格。每个页面都如同一张精心设计的幻灯片，图文布局考究、排版干净、视觉层次分明，旨在为用户提供沉浸式的高级内容创作体验。

该产品通过集成先进的 AI 能力，将繁琐的文字处理、数据结构化、图文排版以及演示文稿制作等工作自动化，大幅降低了高质量内容创作的门槛。无论是自媒体博主、市场营销人员还是日常办公用户，都能通过创作工坊快速生成具有专业设计水准的内容。

## 二、核心功能模块

创作工坊的核心功能围绕内容创作的全链路展开，涵盖了从文字处理到最终视觉呈现的各个环节。为了降低使用门槛，所有核心创作功能（AI 处理、上传、生成）均设计为**无需登录即可使用**，仅在保存项目或查看历史记录时需要用户登录。

| 功能模块 | 路由路径 | 功能说明 |
| :--- | :--- | :--- |
| **首页** | `/` | 提供产品介绍、核心功能入口以及多种 PPT 风格的直观展示，引导用户快速开始创作。 |
| **AI 文字编辑** | `/editor` 或 `/editor/:id` | 支持用户上传文字或图片。提供 AI 润色、扩写、缩写、改写等多种处理模式。内置富文本编辑器，支持原文与 AI 处理后内容的直观对比。 |
| **表格生成** | `/table` | 用户输入非结构化的文字数据，AI 自动解析并生成结构化表格，支持将生成的表格导出为 CSV 格式。 |
| **图文排版** | `/layout` | 用户上传文字和图片素材，AI 自动生成 5 种不同排版风格的 HTML 页面，支持最终效果的导出。 |
| **PPT 生成** | `/ppt` | 提供 6 种预设的美学风格模板，AI 根据用户输入自动生成幻灯片内容，支持导出为 HTML 格式的演示文稿。 |
| **项目管理** | `/projects` | 允许用户查看、继续编辑或删除已保存的创作项目。此功能需要用户登录后方可使用。 |

## 三、特色设计系统

创作工坊在视觉设计上投入了大量精力，建立了一套完整且统一的设计系统，确保所有生成的内容都符合「优雅精致」的定位。

### 1. 全局色彩方案

整体采用「暖奶油 + 深棕金」的配色方案，营造出高级且温暖的视觉感受：
*   **背景色（Background）**：奶油白（`oklch(0.98 0.01 85)`），提供干净柔和的画布。
*   **前景色（Foreground）**：深棕（`oklch(0.2 0.04 60)`），确保文字阅读的舒适度与对比度。
*   **主色调（Primary）**：棕金（`oklch(0.45 0.08 55)`），用于强调核心操作与关键信息。
*   **强调色（Accent）**：玫瑰粉（`oklch(0.85 0.05 30)`），为界面增添灵动与活力。

### 2. 字体排版系统

通过精心挑选的字体组合，提升内容的阅读体验与专业感：
*   **标题字体**：`Playfair Display`（衬线体），赋予标题古典与优雅的气质。
*   **正文字体**：`Lora`（衬线体），保证长文本阅读的流畅性与舒适度。
*   **UI 字体**：`Inter`（无衬线体），用于界面元素，确保信息的清晰传达。

### 3. PPT 美学风格模板

PPT 生成功能内置了 6 种精心设计的美学风格，满足不同场景与主题的创作需求：

| 风格名称 | 视觉描述 | 主色调 | 背景色 |
| :--- | :--- | :--- | :--- |
| **可可特（Coquette）** | 粉色蕾丝蝴蝶结，浪漫少女感 | `#E8A0BF` | `#FFF0F5` |
| **暗黑学院（Dark Academia）** | 深棕暗红，复古书卷气 | `#C4A35A` | `#1C1410` |
| **田园核（Cottagecore）** | 米白花卉，温柔自然感 | `#8B7355` | `#F5F0E8` |
| **极简主义（Minimalism）** | 留白克制，高级线条感 | `#1A1A1A` | `#FAFAFA` |
| **蒸汽波（Vaporwave）** | 霓虹紫粉，赛博复古感 | `#FF71CE` | `#0D0221` |
| **包豪斯（Bauhaus）** | 几何色块，现代构成感 | `#E63946` | `#F5F5F0` |
